Fermionic Hubbard model: (spin-full fermions with interactions).

class tenpy.models.fermions_hubbard.FermionicHubbardChain(model_param)[source]¶

Bases: tenpy.models.model.CouplingModel, tenpy.models.model.NearestNeighborModel, tenpy.models.model.MPOModel

Spin-1/2 fermionic Hubbard model in 1D.

The Hamiltonian reads:

\[H = \sum_{\langle i, j \rangle, \sigma} t (c^{\dagger}_{\sigma, i} c_{\sigma j} + h.c.) + \sum_i U n_{\uparrow, i} n_{\downarrow, i} + \sum_i \mu ( n_{\uparrow, i} + n_{\downarrow, i} ) + \sum_{\langle i, j \rangle, \sigma} V (n_{\uparrow,i} + n_{\downarrow,i})(n_{\uparrow,j} + n_{\downarrow,j})\]

All parameters are collected in a single dictionary model_param and read out with get_parameter().

Parameters:
L : int

Length of the chain

t, U, mu : float | array

Parameters as defined for the Hamiltonian above

cons_N : {‘N’ | ‘parity’ | None}

Whether particle number is conserved, see SpinHalfFermionSite for details.

cons_Sz : {‘Sz’ | ‘parity’ | None}

Whether spin is conserved, see SpinHalfFermionSite for details.

bc_MPS : {‘finite’ | ‘infinte’}

MPS boundary conditions. Coupling boundary conditions are chosen appropriately.
