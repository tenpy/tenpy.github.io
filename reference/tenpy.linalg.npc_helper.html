tenpy.linalg.npc_helper module¶

Optimization of charges.py and np_conserved.py

This module is written in Cython, such that it can be compiled. It implements some functions and classes with the same interface as np_conserved.py/charges.py.

tenpy.tools.optimization.use_cython() tries to import the compiled cython module and uses the functions/classes defined here to overwrite those written in pure Python whenever the decorator @use_cython is used in other python files of tenpy. If this module was not compiled and could not be imported, a warning is issued.

tenpy.linalg.npc_helper.Array__imake_contiguous¶

Make each of the blocks c-style contigous in memory.

Might speed up subsequent tensordot & co by fixing the memory layout to contigous blocks. (No need to call it manually: it’s called from tensordot & co anyways!)
