Module description

Tools to save and load data (from TeNPy) to disk.

The functions dump() and load() are convenience functions for saving and loading quite general python objects (like dictionaries) to/from files, guessing the file type (and hence protocol for reading/writing) from the file ending.

On top of that, this function provides support for saving python objects to [HDF5] files with
