tenpy.models.tf_ising module¶

Prototypical example of a quantum model: the transverse field Ising model.

Like the XXZChain, the transverse field ising chain TFIChain is contained in the more general SpinChain; the idea is more to serve as a pedagogical example for a ‘model’. Choosing the field along z allow to preserve parity, if desired, at the expense of a larger MPO bond dimension for the Hamiltion.

The TFIModel2D contains the same couplings but on a square lattice in 2D (with chooseable boundary conditions). As such, it illustrates the correct usage of the Lattice classes.

Transverse field Ising chain.

The Hamiltonian reads:

\[H = - \sum_{i} \mathtt{J} \sigma^x_i \sigma^x_{i+1} - \sum_{i} \mathtt{g} \sigma^z_i\]

All parameters are collected in a single dictionary model_param and read out with get_parameter().

Parameters:
L : int

Length of the chain

J, g : float | array

Couplings as defined for the Hamiltonian above.

bc_MPS : {‘finite’ | ‘infinte’}

MPS boundary conditions. Coupling boundary conditions are chosen appropriately.

conserve : ‘parity’ | None

What should be conserved. See SpinSite.

class tenpy.models.tf_ising.TFIModel2D(model_param)[source]¶

Bases: tenpy.models.model.CouplingModel, tenpy.models.model.MPOModel

Transverse field Ising model on a square lattice.

The Hamiltonian reads:

\[H = - \sum_{\langle i,j\rangle, i < j} \mathtt{J} \sigma^x_i \sigma^x_{j} - \sum_{i} \mathtt{g} \sigma^z_i\]

Here, \(\langle i,j \rangle, i< j\) denotes nearest neighbor pairs, each pair appearing exactly once. All parameters are collected in a single dictionary model_param and read out with get_parameter().

Parameters:
Lx, Ly : int

Length of the chain in x- and y-direction.

J, g : float | array

Couplings as defined for the Hamiltonian above.

bc_MPS : {‘finite’ | ‘infinte’}

MPS boundary conditions along the x-direction. For ‘infinite’ boundary conditions, repeat the unit cell in x-direction. Coupling boundary conditions in x-direction are chosen accordingly.

bc_y : ‘ladder’ | ‘cylinder’

Boundary conditions in y-direction.

conserve : None | ‘parity’

What should be conserved. See SpinSite.

order : string

Ordering of the sites in the MPS, e.g. ‘default’, ‘snake’; see ordering().
