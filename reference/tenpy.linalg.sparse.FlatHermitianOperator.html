  • full name: tenpy.linalg.sparse.FlatHermitianOperator

class tenpy.linalg.sparse.FlatHermitianOperator(npc_matvec, leg, dtype, charge_sector=0, vec_label=None)[source]¶

Bases: tenpy.linalg.sparse.FlatLinearOperator

Hermitian variant of FlatLinearOperator.

Note that we don’t check matvec() to return a hermitian result, we only define an adjoint to be self.

classmethod from_NpcArray(mat, charge_sector=0)¶

Create an FlatLinearOperator from an square Array.

Parameters
matArray

A square matrix, with contractable legs.

charge_sectorNone | charges | 0

Selects the charge sector of the vector onto which the Linear operator acts. None stands for all sectors, 0 stands for the zero-charge sector. Defaults to 0, i.e., assumes the dominant eigenvector is in charge sector 0.
