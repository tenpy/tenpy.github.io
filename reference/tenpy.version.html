Module description

Access to version of this library.

Before updating a version, make sure nosetest runs all tests in test successfully! To update a version, change version in this module and create a git tag

git tag -a "v1.0.2"

Make shure to push your tag into the shared git repo with git push origin [tagname].

This module provides the three variables version, git_version and full_version. version is a tuple of three integers (major, minor, revision). git_version is a string including information about the current git commit, full_version includes the versions of the used python, numpy and scipy libraries.
