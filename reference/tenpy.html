TeNPy - a Python library for Tensor Network Algorithms¶

TeNPy is a library for algorithms working with tensor networks, e.g., matrix product states and -operators, designed to study the physics of strongly correlated quantum systems. The code is intended to be easy readable for students new to the field, and yet powerful enough for day-to-day research.

The library provides:

  • various predefined models with standard hamiltonians of the field, e.g., the XXZ spin chain
  • various algorithms, e.g., DMRG, TEBD
  • the necessary tools to perform this, including an array class to handle charge conservation for abelian symmetries
